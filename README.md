# College Assignment Submission Tracker

A responsive React web app to manage and track assignment submissions for college courses.

## Features
- Add, edit, delete assignments
- Dashboard with stats and charts
- Search, filter by subject/status
- Dark mode, progress bar, calendar view
- Export to PDF, toast notifications
- Data persisted in localStorage

## Quick start

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm start
```

Build:

```bash
npm run build
```

This is a minimal starter scaffold. Customize styles, add calendar integration, and improve validation as needed.
